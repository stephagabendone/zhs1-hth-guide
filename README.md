# zhs1-hth-guide

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本项目旨在收集、整理并托管一系列独立的 HTML 文件。这些页面彼此独立，不依赖于特定后端或数据库，可直接在浏览器中打开运行。仓库本身不针对任何特定域名或网站，仅作为静态页面的存储与分发场所。

## 目录说明

```
zhs1-hth-guide/
├── pages/          # 存放主要 HTML 页面
├── assets/         # 静态资源文件（CSS、JS、图片等）
├── archive/        # 已归档或不再维护的旧版本页面
└── README.md       # 本文件
```

- **pages/**：当前可用的 HTML 页面，每个文件或子目录代表一个独立页面。
- **assets/**：页面引用的公共资源，按类型或页面分目录存放。
- **archive/**：历史版本或已停止更新的页面，供参考或回溯。

## 页面归档说明

- 每个 HTML 页面均为自包含或依赖本仓库内资源的完整文档。
- 页面可能随时间更新，旧版本会被移入 `archive/` 目录。
- 归档页面保留原始结构，但不再接受功能更新或问题修复。
- 页面命名尽量反映其内容或用途，避免使用模糊或误导性名称。

## 维护说明

- 本仓库由个人维护，更新频率不定。
- 欢迎通过 Issue 提出建议或报告问题，但不保证立即处理。
- 不接受直接向仓库推送未经审核的页面或资源。
- 仓库内容仅作归档与展示用途，不提供任何形式的服务或承诺。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的任意 `.html` 文件。
3. 如需在线访问，可借助 GitHub Pages 等静态托管服务部署本仓库。
